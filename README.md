## 🧭 What this is

climb-the-tall-building-0 is a card game for Windows. You build a deck, face enemies, and climb through a tall building one floor at a time.

Each run is different. You pick cards, shape your deck, and try to go farther than before. The game uses a clean browser-style interface and runs as a desktop app.

## 🎴 How the game works

You begin with a small deck. Your cards let you attack, defend, and set up stronger turns.

As you move up the building, you face:

- Basic enemies
- Stronger enemies
- Reward screens
- New card choices
- Risk and reward decisions

The goal is simple: build a deck that can handle harder fights and climb as high as you can.

## 🧠 Tips for new players

- Take cards that fit your deck plan
- Do not add every card you see
- Keep your deck focused
- Use defense cards before big enemy attacks
- Watch enemy moves before you act
- Try different card mixes after each run
- Build around cards that work well together

A typical run feels like this:
